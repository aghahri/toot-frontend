چه چیزهایی عوض شد؟ (بدون تغییر IA یا route)

  • Navbar از لینک‌های انگلیسی Home/Profile/Upload + Logout پاک شد؛ فقط برند «توت» و آواتار ماند.
  • Bottom nav از پیل پرشدهٔ bg-slate-900 به بولد + نقطهٔ لهجه تغییر کرد. آیکن ۲۰px اضافه شد. بج نخوانده روی «گفتگو».
  • /home: نوار استوری افقی، کارت‌های فید با ترتیب RTL درست (آواتار → نام → handle → زمان → منو)، اکشن‌ها با اعداد فارسی، FAB نوشتن پایین-چپ.
  • /direct/[id]: Navbar جای خود را به هدر چت داد (back + آواتار + حضور). حباب‌ها با واریاسیون گوشه (بدون تغییر رنگ سیاه سنگین)، تایم‌استمپ داخل حباب، جداکنندهٔ روزانه، کامپوزر چسبان با textarea و آیکن ارسال گرد.
  • /vitrin از کارت توضیحی «چارچوب محصول» به گرید واقعی ۲ستونه رسید — تصویر ۴:۵، عنوان، فروشنده، قیمت فارسی با «تومان» بعد از عدد.
  • /spaces همینطور — هیرو محلی + نوار چیپ فیلتر + گرید کارت فضا با کاور، آواتار، تعداد عضو.
  • پس‌زمینه از slate-50 سرد به کاغذی گرم #F7F7F5 رفت. لهجهٔ واحد #B4532A (چای). فونت Vazirmatn.
  • همهٔ کامپوننت‌ها و state های موجود (reply, typing, delete, media preview, unread, loading) قابل‌انطباق‌اند — فقط کلاس‌ها عوض می‌شوند.
